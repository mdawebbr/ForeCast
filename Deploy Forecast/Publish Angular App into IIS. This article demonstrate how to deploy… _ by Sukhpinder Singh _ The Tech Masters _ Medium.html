AuthGuard Authentication

Auth guard is an angular route guard that is used to stop unauthenticated or unauthorized users from accessing the protected routes. — It is implemented using the “CanActivate” interface, which allows the guard to decide if a way can be activated with the “CanActivate()” method. If the method returns “true,” then the route is activated(allowed to proceed). If the method returns “false,” then it will be redirected to the login page.
